# 個人記帳小工具（純前端版）

這是一個使用 **HTML + CSS + 原生 JavaScript** 實作的輕量記帳小工具，只記錄「支出」，並將資料儲存在瀏覽器的 `localStorage`，重新整理或關閉瀏覽器後資料都會保留在本機。

## 功能介紹

- **只記錄支出**
  - 每筆支出包含：金額、分類（食物 / 交通 / 娛樂 / 購物 / 其他）、日期、備註。
- **支出統計**
  - 顯示「本月總支出」。
  - 顯示「今日支出」。
- **支出列表**
  - 依日期由新到舊排序。
  - 顯示金額、分類、日期與備註。
  - 可刪除單筆記錄。
- **主題色系切換**
  - 右上角可切換 4 種色系：
    - 櫻花粉（預設）
    - 薄荷綠
    - 天空藍
    - 薰衣草紫
  - 主題選擇會儲存在 `localStorage`，重整後仍會維持。
- **RWD 響應式介面**
  - 手機與桌面版都有針對性排列與間距調整。

## 介面布局

- **桌面版（寬度 > 768px）**
  - 左側：統計卡片（本月 / 今日）＋ 新增支出表單。
  - 右側：支出記錄列表。
  - 左右欄位比例約 4:6，桌面上可以同時看到表單與記錄，不用上下捲動切換。
- **手機版（寬度 ≤ 768px）**
  - 統計卡片：兩張卡片左右並排。
  - 新增表單：單欄排列，按鈕全寬。
  - 支出記錄：出現在表單下方，整頁自然滾動。

## 檔案結構

- `index.html`：主頁面結構，包含：
  - 標題與主題切換下拉選單。
  - 統計卡片區塊。
  - 新增支出表單。
  - 支出記錄列表。
- `style.css`：整體視覺樣式與 RWD 設計，包含：
  - 櫻花粉 / 薄荷綠 / 天空藍 / 薰衣草紫 四種主題變數。
  - 卡片、按鈕、表單欄位、列表項目等樣式。
  - 手機與桌面版的版面配置。
- `app.js`：全部的前端邏輯，包含：
  - 新增支出、刪除支出。
  - 計算本月總支出與今日支出。
  - 使用 `localStorage` 儲存與載入支出資料。
  - 主題色系切換與主題設定的儲存 / 載入。

## 使用方式

1. 將整個資料夾放在任意位置。
2. 以瀏覽器開啟 `index.html`（直接拖曳到瀏覽器或使用右鍵「以瀏覽器開啟」）。
3. 在「新增支出」表單中輸入：
   - 金額（必填，需為正數）
   - 分類（必填）
   - 日期（必填，預設為今天）
   - 備註（選填）
4. 點擊「新增支出」後，支出會出現在右側列表，上方統計會自動更新。
5. 需要刪除某筆支出時，按該筆右側的「刪除」按鈕即可。

## 瀏覽器支援

- 主要以現代瀏覽器為目標（Chrome / Edge / Firefox / Safari 等）。
- 需要支援：
  - `flex` / `grid` / `CSS variables`
  - `localStorage`
  - 原生 ES5+ JavaScript
