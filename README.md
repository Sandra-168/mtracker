# 個人記帳小工具（純前端版）

這是一個使用 **HTML + CSS + 原生 JavaScript** 實作的輕量記帳小工具，只記錄「支出」，並將資料儲存在瀏覽器的 `localStorage`，重新整理或關閉瀏覽器後資料都會保留在本機。

## 功能介紹

- **只記錄支出**
  - 每筆支出包含：金額、分類（食物 / 交通 / 娛樂 / 購物 / 其他）、日期、備註。
- **支出統計**
  - 顯示「本月總支出」。
  - 顯示「今日支出」。
- **支出列表**
  - 依日期由新到舊排序。
  - 顯示金額、分類、日期與備註。
  - 可刪除單筆記錄。
- **日期範圍與分類篩選**
  - 可設定「起始日期」與「結束日期」，只顯示特定期間的支出。
  - 可依分類（全部 / 食物 / 交通 / 娛樂 / 購物 / 其他）篩選。
  - 篩選條件會同時套用在列表、統計總額與分類長條圖。
- **依分類統計長條圖**
  - 以簡易長條圖顯示目前篩選條件下，各分類的總支出。
  - 同時顯示「目前篩選總額」。
- **主題色系切換**
  - 右上角可切換 4 種色系：
    - 櫻花粉（預設）
    - 薄荷綠
    - 天空藍
    - 薰衣草紫
  - 主題選擇會儲存在 `localStorage`，重整後仍會維持。
- **匯出 / 匯入支出資料**
  - 匯出所有支出為 `CSV` 檔。
  - 匯出所有支出為 `JSON` 檔。
  - 支援匯入 `CSV` 或 `JSON` 檔，會將檔案中的紀錄 **加入** 目前資料。
- **本月預算與超支提醒**
  - 可設定本月預算金額，儲存在 `localStorage`。
  - 顯示本月已花金額、剩餘金額與預算使用率。
  - 若本月支出超過預算，會以紅色文字顯示「已超出本月預算 X 元！」。
- **RWD 響應式介面**
  - 手機與桌面版都有針對性排列與間距調整。

## 介面布局

- **桌面版（寬度 > 768px）**
  - 左側：統計卡片（本月 / 今日）＋ 新增支出表單。
  - 右側：支出記錄列表。
  - 左右欄位比例約 4:6，桌面上可以同時看到表單與記錄，不用上下捲動切換。
- **手機版（寬度 ≤ 768px）**
  - 統計卡片：兩張卡片左右並排。
  - 新增表單：單欄排列，按鈕全寬。
  - 支出記錄：出現在表單下方，整頁自然滾動。

## 檔案結構

- `index.html`：主頁面結構，包含：
  - 標題與主題切換下拉選單。
  - 統計卡片區塊。
  - 本月預算設定卡片。
  - 新增支出表單。
  - 支出記錄列表。
  - 日期範圍 / 分類篩選欄位與工具列（匯出、匯入、篩選總額、分類長條圖）。
- `style.css`：整體視覺樣式與 RWD 設計，包含：
  - 櫻花粉 / 薄荷綠 / 天空藍 / 薰衣草紫 四種主題變數。
  - 卡片、按鈕、表單欄位、列表項目、篩選區、預算區、分類長條圖容器等樣式。
  - 手機與桌面版的版面配置。
- `app.js`：全部的前端邏輯，包含：
  - 新增支出、刪除支出。
  - 計算本月總支出與今日支出。
  - 使用 `localStorage` 儲存與載入支出資料。
  - 主題色系切換與主題設定的儲存 / 載入。
  - 日期範圍 / 分類篩選邏輯，並同步更新列表、篩選總額與分類長條圖。
  - 使用 `canvas` 畫出依分類支出的長條圖。
  - 匯出 / 匯入 CSV / JSON 的解析與處理。
  - 本月預算的儲存 / 載入與超支提醒邏輯。

## 使用方式

1. 將整個資料夾放在任意位置。
2. 以瀏覽器開啟 `index.html`（直接拖曳到瀏覽器或使用右鍵「以瀏覽器開啟」）。
3. 在「新增支出」表單中輸入：
   - 金額（必填，需為正數）
   - 分類（必填）
   - 日期（必填，預設為今天）
   - 備註（選填）
4. 點擊「新增支出」後，支出會出現在右側列表，上方統計會自動更新。
5. 需要刪除某筆支出時，按該筆右側的「刪除」按鈕即可。
6. 可在右側列表上方設定「起始日期 / 結束日期 / 分類」進行篩選，分類長條圖與「目前篩選總額」會跟著更新。
7. 可在「本月預算」卡片輸入金額並儲存，之後每次新增支出都會重新計算剩餘金額與是否超支。
8. 可使用「匯出 CSV / 匯出 JSON」按鈕下載目前所有支出資料；使用「匯入檔案」可將 CSV 或 JSON 檔加入現有資料。

## 瀏覽器支援

- 主要以現代瀏覽器為目標（Chrome / Edge / Firefox / Safari 等）。
- 需要支援：
  - `flex` / `grid` / `CSS variables`
  - `localStorage`
  - 原生 ES5+ JavaScript

## 後續可擴充方向（建議）

- 更多圖表形式（例如：月視圖趨勢線、每週支出折線圖）。
- 支援多種貨幣與匯率換算。
- 匯出時提供自訂欄位/範圍選項。
- 加入「月目標」或「分類預算」，例如針對「娛樂」單獨限制金額。
